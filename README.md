 # Library Management System (C++)

A **fully functional console-based library system** using **Object-Oriented Programming (OOP)** in C++.

---

## Features
- User Registration & Login
- Book Management (Add, Update, Remove)
- Search by ISBN or Title
- Persistent Storage (`users.txt`)
- Clean & Commented Code
